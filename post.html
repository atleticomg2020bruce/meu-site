O ZArchiver é uma aplicação que permite administrar qualquer pasta compactada no seu dispositivo Android. Não importa se quiser ver o interior de um documento comprimido, ou simplesmente quer criar um arquivo comprimido utilizando diversos arquivos do seu cartão SD, o ZArchiver fará tudo em apenas alguns segundos.

O programa dá a opção de criar arquivos comprimidos em qualquer um dos seguintes formatos: 7z (7zip), zip, bzip2 (bz2), gzip (gz), XZ, tar. Além de lhe permitir descomprimir ficheiros que vêm em 7z, zip, rar, bzip2, gzip, XZ, iso, tar, arj, cab, lzh, lha, lzma, xar, tgz, tbz, Z, deb, rpm, zipx, mtz. Com o ZArchiver também é possível ver o conteúdo de arquivos nessas extensões, mesmo que estejam protegido com senha.

O ZArchiver é um compactador obrigatório para qualquer usuário que necessite trabalhar com este tipo de arquivos no seu dispositivo Android.

Perguntas frequentes

Posso comprimir um arquivo com ZArchiver?

Para comprimir arquivos com ZArchiver, tudo o que você precisa fazer é selecionar os arquivos que você deseja, clicar nos três pontos no canto superior direito, e então em "Compress." Depois disso, escolha o tipo de compressão e o nome que você deseja dar a ele.

Como uso ZArchiver?

Usar ZArchiver é bastante simples, pois navegar nele é semelhante a navegar em outros gerenciadores de arquivos. Assim que você instalar o app, você poderá abrir todos os tipos de arquivos, bem como compactar ou descompactar qualquer arquivo que desejar.

ZArchiver precisa de uma conexão de internet?

Não é necessário ter conexão de internet para usar os recursos de ZArchiver. Você pode usar este gerenciador de arquivos offline, sem a utilização de WiFi ou dados móveis.

Como eu descompacto um arquivo com ZArchiver?

Para descompactar um arquivo com ZArchiver, selecione o arquivo comprimido, e então clique em "Extract" para acessar os arquivos dentro dele. Existem diferentes opções de descompactação, então você pode personalizar este processo de acordo com as suas necessidades.
